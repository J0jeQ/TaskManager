# TaskManager
Task Manager to aplikacja backendowa umożliwiająca zarządzanie zadaniami. Aplikacja pozwala użytkownikom na tworzenie, usuwanie oraz wyświetlanie zadań. Jest to prosta aplikacja, która może być używana jako baza do dalszego rozwijania systemów zarządzania zadaniami.

## Technologie

- **Node.js**
- **Express**
- **MongoDB**
- **Mongoose**

## Funkcjonalności

API oferuje następujące funkcje:

1. **Pobierz wszystkie zadania** - Endpoint, który zwraca wszystkie zapisane zadania.
2. **Pobierz zadanie po ID** - Endpoint umożliwiający pobranie zadania na podstawie jego unikalnego ID.
3. **Dodaj nowe zadanie** - Endpoint do tworzenia nowych zadań.
4. **Usuń zadanie** - Endpoint do usuwania zadania.
